# 篇章逻辑一致性独立核验报告

判定口径：

- PASS：核心不一致识别完整；干净稿无误报；建议不替用户裁定未知事实。
- WARN：核心问题识别正确，但修改建议存在轻微事实选择倾向。
- FAIL：漏掉核心不一致、明显误报，或直接补造事实、改变用户原意。

## 逐项判定

| 用例 | Reviewer A | Reviewer B | 核验结论 |
|---|---|---|---|
| P1 长文干净对照 | PASS | PASS | 原稿各组数量、状态、抽样范围、阶段安排和字段库待审议状态能够对应。两人均未误报。 |
| P2 长文摘要与正文 | PASS | WARN | 两人均完整发现任务完成状态、74个问题状态、字段库覆盖状态、常态化运行与后续程序四类远距离冲突。A主要使用条件式或要求依据原始状态统一，没有替用户选数。B多次直接要求“摘要采用正文口径”，虽然正文证据更完整，但仅凭稿内冲突仍不能确定哪一处代表真实状态，修改建议存在替用户选择冲突事实的倾向。 |
| P3 长文结论与正文 | PASS | WARN | 两人均发现132个事项状态、74个问题状态、字段库推广条件和后续程序四类冲突。A保留了“依据原始材料核定”或条件式处理。B直接要求删除结论并采用正文状态，审稿判断正确，但修改建议没有保留核定真实状态这一层。 |
| P4 隐含因果 | PASS | PASS | 两人均识别同期存在工具上线、审核层级调整和增员三项变化，现有数据不能确认工具的单独直接作用；也识别全面推广范围超过现有证据。没有把单纯缺少细节误报为矛盾。 |
| P5 样本范围与总体结论 | PASS | PASS | 两人均发现样本外推、问题性质和影响强度升级、措施范围超过证据三类问题，并确认32次记录与6、3、23的数量关系自洽。 |
| P6 决策状态修改联动 | PASS | PASS | 两人均发现合同签订、部署、上线准备和末段立即采购结论仍残留，完整覆盖用户最新修改要求。删除建议有明确用户指令依据，不属于擅自改变事实。 |
| P7 责任主体修改联动 | PASS | PASS | 两人均发现“信息技术部汇总”残留，与用户要求统一改为数据管理处冲突。未误改OA渠道、截止时间或其他内容。 |
| P8 短稿干净对照 | PASS | PASS | 接收对象、渠道、日期、汇总责任、职责边界、联系人和落款一致。两人均未误报。 |
| P9 修改联动干净对照 | PASS | PASS | 新责任单位和新期限已经全文联动，旧单位和旧期限没有残留。两人均未误报。 |

## Reviewer 实际失败聚合

以下只统计 reviewer 自身的 WARN、FAIL，不统计样稿人工设置的问题。

| 风险类别 | WARN | FAIL | 分布 | 是否达到门槛 |
|---|---:|---:|---|---|
| L 远距离摘要/结论与正文不一致 | 2 | 0 | Reviewer B：P2、P3 | 否。虽覆盖2个 prompt，但只有2份 reviewer 输出，且只涉及1名 reviewer。 |
| E 证据—因果—范围—措施跳跃 | 0 | 0 | 无 | 否 |
| R 最新修改要求与全文联动残留 | 0 | 0 | 无 | 否 |
| C 干净对照误报 | 0 | 0 | 无 | 否 |

没有任何一类同时达到“至少3份 reviewer 输出失败、覆盖至少2个不同 prompt、涉及2名独立 reviewer”的门槛。WARN 与 FAIL 均未混算；本轮没有 reviewer FAIL。

## 是否触发 Prompt 修改门槛

不触发在 `proofreading-checklist.md` 增加短小节的门槛。

当前两名 reviewer 已稳定识别：

- 摘要、结论与远距离正文的状态和程序冲突；
- 多因素并行情况下的直接因果越界；
- 样本范围、结论强度和措施范围的跳跃；
- 决策状态、责任主体修改后的全文残留；
- 干净稿中的合理职责分工，并未因缺少更多细节而误报。

Reviewer B 的事实选择倾向值得保留为后续观察点，但只有同一 reviewer 在P2、P3出现两次WARN，尚不足以形成共性产品缺陷。后续评测可在审稿要求中继续观察“指出冲突，但不默认正文、摘要或结论任一处为真”，暂不修改产品 Prompt。

## 本轮还不能证明

- 不能证明3000字以上、多附件、多文档交叉引用中的远距离一致性稳定性；P2、P3来自同一份长文及同类状态冲突。
- 不能证明多跳因果、条件例外、概念偷换、对象范围缓慢漂移等更隐蔽问题的检出能力。
- 不能证明大范围结构调整后的联动能力；P6、P7主要是显式决策句和责任单位残留。
- 不能证明实际改稿后的闭环质量；本轮只审不改，没有生成修订稿并再次核验数字、状态、责任和期限是否全部同步。
- 不能以三个干净对照推导稳定误报率。
- 不能证明外部事实正确、政策依据有效或业务决策合理；本轮只检验稿内一致性。
