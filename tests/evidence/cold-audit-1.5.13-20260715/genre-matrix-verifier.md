# 文种矩阵独立 Verifier

## 输入与判定方法

本轮只读取 `genre-matrix-writer-c.md` 与 `genre-matrix-writer-d.md`。第一阶段忽略每题 `ROUTE` 行，仅依据原始 prompt 与成稿判定内容；内容判分锁定后，第二阶段才统计 `ROUTE` 自报。

判定重点包括用户指令、数字、主体、对象、日期、期限、责任、表决、附件和正文外缺项，文种功能与行文关系，禁止事项，新增事实，Markdown、过程说明和保护性自证。`PASS` 表示未发现影响指令遵循、事实边界或文种功能的问题；`WARN` 表示存在非阻断瑕疵；`FAIL` 表示存在明确违背。

## 证据限制

本轮未读取任何 Skill 源码、冷审报告、baseline verifier、其他 evidence、tests 或网络资料。以上 ROUTE 统计只能证明两个 packet 中写明了哪些自报项，不能证明系统实际访问、加载或未加载了哪些文件，也不能作为访问审计。本报告只判定这28份原 prompt 与成稿，不外推到其他模型、其他 writer、其他文种或整体版本质量，也不据此提出修改建议。
