## 通过口径

- 事实、文种、格式和输出模式先判 PASS/WARN/FAIL；Skill 独有 FAIL 或实质性 WARN 记硬回退。
- 篇幅同时记录 CJK 汉字数和去空白可见字符数，由 verifier 结合中文“字数”习惯判断；机械计数不代替整稿判断。
- 质量比较覆盖信息利用、首中尾均衡、段落推进、主体宾语、结论强度、语言自然度、固定句式和直接采用成本。
- 只有 Skill 硬边界不低于无 Skill、整稿明确更好时才记严格胜出。平局、分歧、共同失败和证据不足均不计。
- 本文件和 writer prompt 在候选修改后不得改写；如发现任务本身有矛盾，该任务整组三模型作废并单独说明。
