# 机械测量

机械测量用于辅助定位篇幅和显式句式，不替代独立 verifier 的事实与写作质量判断。

## 非空白字符

| 模型与条件 | T01 | T02 | T03 | T04 |
| --- | ---: | ---: | ---: | ---: |
| LUNA 无 Skill | 1473 | 1408 | 613 | 931 |
| LUNA 带 Skill，正常路由 | 1308 | 1337 | 577 | 647 |
| LUNA 带 Skill，强制完整路由 | 988 | 979 | 416 | 512 |
| TERRA 无 Skill | 1560 | 1336 | 705 | 1023 |
| TERRA 带 Skill，正常路由 | 1406 | 1107 | 603 | 903 |

四题要求依次为 1600—1900、1300—1600、700—900、900—1100 字。非空白字符不是精确中文成稿字数，但同一计数口径下，两个模型的正常带 Skill 稿都逐题短于各自无 Skill 稿；LUNA 强制完整路由又逐题短于正常带 Skill 稿。

## 显式句式

按正则 `不是.{0,30}而是` 扫描，本轮 20 份稿均为 0。因而本轮不能把“减少不是……而是……”作为真实写稿对比结论。

无 Skill 与带 Skill 稿均使用了与材料未决状态相符的否定词。单纯计数不能区分必要状态、保护性说明和无锚定否定，应以逐句盲审为准。
