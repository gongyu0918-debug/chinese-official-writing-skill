# 三稿匿名盲审包

三份稿件来自同一写作模型、相同 thinking、相同原始任务和相同宿主外壳下的独立上下文。来源标签已匿名化；只依据原始任务和成稿判断，不猜测来源，也不调用或读取任何公文写作 Skill。

## 裁决顺序

1. 先逐稿核对硬边界：事实与数字、主体与对象、未决状态和结论强度、文种与行文关系、用户格式或模板、Prompt 动作、输出模式。每项给 PASS/WARN/FAIL，并引用具体句子。
2. 用户明示篇幅范围时，篇幅只作为 Prompt 遵循项单列。两稿都在范围内时，长短差异退出质量比较；更接近区间中点不加分。两稿共同不合规时，不因某稿“更接近”而判优。篇幅合规不能单独构成胜出。
3. 再比较非篇幅质量：结构与信息利用、首中尾均衡、主体和宾语锚定、段落推进、可读性、自然度、机械句式和直接使用成本。信息写得更多本身不加分；材料外扩写不算信息利用。
4. “先……再……”“一是……二是……”或正式连接词单次出现不算机械。只有成簇重复、没有真实关系的连接词链、每段同构、边界自证循环、固定三段式、口号式收束或主体宾语反复缺失时，才判机械，并至少引用两处证据。
5. 直接使用成本分为：G0 可直接采用；G1 只需不超过 3 处局部修正；G2 需要重组一段以上、补结构要素或系统删重；G3 事实、文种、格式或输出模式失真，需要重写。范围内的篇幅差异不改变成本等级。
6. 给出三稿排序。第一名相对第二名只可判“严格胜出”“小胜”或“难分”。“严格胜出”要求硬边界不低、至少两个非篇幅质量优势、其中至少一项来自可读性/自然度/直接使用成本，并且没有实质质量维度更差。
7. 若一稿事实更稳而另一稿更自然，或裁决只能依赖篇幅、偏好或单句差异，判为难分，不强行分胜负。

## 输出格式

- 硬边界表：A/B/C 的逐项 PASS/WARN/FAIL 与原文证据。
- 质量比较表：各维度谁更好或相当、文本证据、采用影响。
- 排序：第一、第二、第三；第一相对第二为严格胜出/小胜/难分。
- 风险：是否存在任何一稿独有的事实、状态、文种、格式、Prompt 或输出模式硬回退。

### 稿件 A

数据中心机房制冷改造专题会会议纪要

2026年7月9日14时30分，周明在第二会议室主持召开数据中心机房制冷改造专题会。办公室、信息中心、财务处、后勤处参加会议。会议听取了信息中心、后勤处和财务处有关情况报告，围绕机房制冷设备运行状况、改造必要性、资金安排及后续论证事项进行了研究。

会议指出，5月1日至6月30日期间，机房共出现9次高温告警，其中7次发生在13时至17时，最高温度达到31.8℃，高温告警在午后时段较为集中。现有3台精密空调中，1台已运行满8年，另外2台已运行满5年，设备运行年限及告警情况需要结合负荷状况进一步分析。信息中心要根据现有监测情况，客观反映机房制冷设备运行状态，为后续论证提供依据。

后勤处报告，6月20日已完成一次设备清洗和冷媒检查，6月21日至30日期间仍出现2次高温告警。会议认为，清洗和冷媒检查后告警次数有所减少，但目前掌握的时间和数据不足以据此确定长期效果，不能简单将此次告警次数变化作为判断设备是否需要改造、改造范围如何确定的唯一依据。有关情况还需结合负荷测算、设备运行年限及高温时段告警情况综合研判。

财务处报告，本年度设备更新预算尚余48万元。目前，两家供应商初步报价分别为45.6万元和52.3万元，但两家报价所包含的范围不同，暂不能直接进行价格比较。会议认为，现阶段资金情况可以作为后续论证的重要参考，但在改造范围、设备配置及报价内容尚未统一的情况下，不能仅依据报价金额作出供应商选择或项目实施结论。

会议提出，下一步应先补齐负荷测算、施工窗口和业务连续性安排等基础信息，再提交馆长办公会研究。经会议研究，一致同意于7月18日前完成负荷测算，由信息中心和后勤处共同提供基础数据。负荷测算完成后，再结合设备运行情况和既有告警记录，对制冷改造的必要性及范围进行进一步分析。

会议明确，当前对采用整体更换还是分步改造、是否在本年度实施、选择哪家供应商等事项，均未作出决定。施工窗口、停机安排、采购方式和验收指标也尚未确定。上述事项应在基础数据和相关方案进一步明确后，再按程序研究。有关单位要准确区分已形成的共识事项与尚待研究事项，避免将阶段性分析意见表述为最终决定。

会议要求，信息中心、后勤处按照会议确定的时间要求，共同完成负荷测算所需基础数据的提供和整理，并于7月18日前形成测算结果。后续是否实施改造、采取何种改造方式、安排何时施工、采用何种采购方式及设置何种验收指标，待相关基础工作完成后，提交馆长办公会研究。

### 稿件 B

数据中心机房制冷改造专题会会议纪要

2026年7月9日14时30分，周明主持召开数据中心机房制冷改造专题会，办公室、信息中心、财务处、后勤处有关人员参加会议。会议听取了信息中心、后勤处和财务处有关情况汇报，就机房制冷设备运行状况、改造必要性、预算安排及后续论证工作进行了研究。现纪要如下：

一、关于机房运行情况。信息中心报告，5月1日至6月30日期间，机房共出现9次高温告警，其中7次发生在13时至17时，期间最高温度达到31.8℃。目前机房配置3台精密空调，其中1台已运行满8年，另有2台已运行满5年。会议认为，现有设备运行年限较长，且高温告警主要集中在业务运行时段，机房制冷保障能力和设备运行稳定性需要进一步评估。

二、关于既有设备维护情况。后勤处报告，6月20日已完成一次设备清洗和冷媒检查，但6月21日至30日期间仍发生2次高温告警。现阶段只能确认维护后告警次数有所减少，尚不能据此判断维护措施的长期效果，也不能直接作为确定改造方案的依据。会议要求结合设备运行记录、环境温度、负荷变化和维护前后数据，对告警原因及维护效果作进一步分析。

三、关于资金和供应商报价情况。财务处报告，本年度设备更新预算尚余48万元。目前已取得两家供应商初步报价，分别为45.6万元和52.3万元。由于两家报价所涵盖的设备配置、施工内容、服务期限及其他条件不同，暂不能直接进行价格比较。会议要求后续在统一技术需求和报价边界的基础上，进一步核实项目投资规模、资金使用条件及预算承受能力。

四、关于后续论证安排。办公室提出，应先补齐负荷测算、施工窗口和业务连续性安排等基础论证材料，再提交馆长办公会研究。会议对此表示同意，明确由信息中心、后勤处共同负责基础数据收集和负荷测算工作，于7月18日前完成并形成阶段性成果。负荷测算应充分考虑现有设备运行状态、机房实际负荷、峰值时段、冗余需求及未来业务发展等因素，为方案比选提供依据。

五、关于事项边界。会议指出，现阶段仅就开展前期论证和补充基础资料形成一致意见，对采用整体更换还是分步改造、是否在本年度实施、选择哪家供应商等事项，均未作决定。施工窗口、停机安排、采购方式和验收指标等内容，也需待技术方案、业务连续性措施及预算情况进一步明确后研究确定。有关部门不得将初步报价、维护情况或本次会议意见作为项目实施、采购定标或供应商选择的依据。

会议要求，各相关单位按照职责分工推进资料收集和论证工作，确保数据真实、口径一致、依据充分。信息中心负责提供告警记录、设备运行数据和业务负荷基础资料；后勤处负责提供设备维护记录、现场条件及施工组织建议；财务处负责配合核实预算及报价边界；办公室负责汇总相关材料，待论证成果形成后按程序提交馆长办公会研究。
