# 三稿匿名盲审包

三份稿件来自同一写作模型、相同 thinking、相同原始任务和相同宿主外壳下的独立上下文。来源标签已匿名化；只依据原始任务和成稿判断，不猜测来源，也不调用或读取任何公文写作 Skill。

## 裁决顺序

1. 先逐稿核对硬边界：事实与数字、主体与对象、未决状态和结论强度、文种与行文关系、用户格式或模板、Prompt 动作、输出模式。每项给 PASS/WARN/FAIL，并引用具体句子。
2. 用户明示篇幅范围时，篇幅只作为 Prompt 遵循项单列。两稿都在范围内时，长短差异退出质量比较；更接近区间中点不加分。两稿共同不合规时，不因某稿“更接近”而判优。篇幅合规不能单独构成胜出。
3. 再比较非篇幅质量：结构与信息利用、首中尾均衡、主体和宾语锚定、段落推进、可读性、自然度、机械句式和直接使用成本。信息写得更多本身不加分；材料外扩写不算信息利用。
4. “先……再……”“一是……二是……”或正式连接词单次出现不算机械。只有成簇重复、没有真实关系的连接词链、每段同构、边界自证循环、固定三段式、口号式收束或主体宾语反复缺失时，才判机械，并至少引用两处证据。
5. 直接使用成本分为：G0 可直接采用；G1 只需不超过 3 处局部修正；G2 需要重组一段以上、补结构要素或系统删重；G3 事实、文种、格式或输出模式失真，需要重写。范围内的篇幅差异不改变成本等级。
6. 给出三稿排序。第一名相对第二名只可判“严格胜出”“小胜”或“难分”。“严格胜出”要求硬边界不低、至少两个非篇幅质量优势、其中至少一项来自可读性/自然度/直接使用成本，并且没有实质质量维度更差。
7. 若一稿事实更稳而另一稿更自然，或裁决只能依赖篇幅、偏好或单句差异，判为难分，不强行分胜负。

## 输出格式

- 硬边界表：A/B/C 的逐项 PASS/WARN/FAIL 与原文证据。
- 质量比较表：各维度谁更好或相当、文本证据、采用影响。
- 排序：第一、第二、第三；第一相对第二为严格胜出/小胜/难分。
- 风险：是否存在任何一稿独有的事实、状态、文种、格式、Prompt 或输出模式硬回退。

（一）短时离线原因尚需进一步查明。4月12日、5月7日、6月18日，设备分别出现一次短时离线，持续11分钟、24分钟、9分钟。前两次经重启恢复，第三次在网络恢复后自行恢复。三次离线均未造成长期业务中断，但目前尚未形成对共同原因的明确结论。有关网络环境、设备终端、后台服务及处置记录之间的关联情况，仍需结合后续运行数据进一步分析，避免仅依据个别现象作出判断。

（二）读者操作体验仍有改进空间。试运行期间共收到读者反馈87条。其中，操作提示不清31条，占比最高；少儿图书识别失败19条；预约书取书位置不易查找16条；希望延长夜间服务13条；其他8条。上述反馈反映出设备基本功能已能满足部分读者需求，但操作界面提示、少儿馆藏识别适配、预约书取书引导等环节仍需优化。夜间服务需求也值得在后续运行中结合实际使用量和保障条件进一步评估。

（三）业务日志时间一致性需持续关注。6月抽查120笔记录，发现7笔日志时间与业务时间相差1至3分钟，其余113笔一致。现有抽查结果显示，大部分记录时间能够对应实际业务时间，但少量偏差提示设备、网络或后台系统的时间同步情况仍应持续监测。后续应加强日志核验，确保业务记录的完整性、可追溯性和统计口径的一致性。

（四）高峰时段运行情况尚需补充测试。本轮试运行已形成一定业务量，但对晚间、高峰时段以及不同点位高并发使用场景的观察仍不充分。特别是设备连续使用、人工窗口协同、现场引导和异常处置等情况，需要通过延长运行和针对性记录进一步验证，为后续是否调整设备数量提供依据。

四、下一步工作建议

根据6月26日馆务会听取阶段汇报后的意见，建议继续运行至8月31日，在保持现有试运行范围和基本保障方式的基础上，重点做好以下工作。

一是补充高峰时段和晚间使用数据。围绕晚间服务需求、不同点位使用量、设备连续运行情况等开展记录，形成可比较的数据，为评估服务时段和设备配置提供参考。

二是持续跟踪少儿图书识别情况。对识别失败的具体图书类型、发生频次、设备状态和处理结果进行记录，区分偶发问题与重复问题，研究是否需要优化识别参数、图书加工方式或现场引导方式。

三是优化预约书取书引导。针对读者反映预约书取书位置不易查找的问题，结合现场标识、设备提示和工作人员引导等方式进行改进，并持续收集读者反馈，检验改进效果。

四是建立离线情况专项记录机制。对离线发生时间、持续时长、设备状态、网络状态、恢复方式和处置过程进行完整记录，推动形成可供分析的连续资料，尽快查明三次短时离线是否存在共同原因。

五是加强运行数据和日志核验。继续开展业务记录抽查，关注日志时间与业务时间的一致性；对发现的时间偏差进行分类登记，核实产生环节和影响范围，保障后续评估依据准确可靠。

六是做好阶段评估准备。试运行延长期间，重点汇总晚间使用量、少儿图书识别情况、预约书取书路径反馈和离线情况等资料。待补充高峰时段测试后，再对设备运行成效、服务需求、经费情况和风险因素进行综合评估，研究是否扩大设备数量。

目前，具体设备调整方案、责任分工和新增采购安排均尚未确定。按照馆务会意见，后续应以补充测试和运行数据为基础，审慎研究有关事项。本阶段不建议将试运行结论直接作为新增采购依据，也不宜在未完成相关测试和评估前作出扩大设备数量的决定。
