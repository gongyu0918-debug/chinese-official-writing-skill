# 三稿匿名盲审包

三份稿件来自同一写作模型、相同 thinking、相同原始任务和相同宿主外壳下的独立上下文。来源标签已匿名化；只依据原始任务和成稿判断，不猜测来源，也不调用或读取任何公文写作 Skill。

## 裁决顺序

1. 先逐稿核对硬边界：事实与数字、主体与对象、未决状态和结论强度、文种与行文关系、用户格式或模板、Prompt 动作、输出模式。每项给 PASS/WARN/FAIL，并引用具体句子。
2. 用户明示篇幅范围时，篇幅只作为 Prompt 遵循项单列。两稿都在范围内时，长短差异退出质量比较；更接近区间中点不加分。两稿共同不合规时，不因某稿“更接近”而判优。篇幅合规不能单独构成胜出。
3. 再比较非篇幅质量：结构与信息利用、首中尾均衡、主体和宾语锚定、段落推进、可读性、自然度、机械句式和直接使用成本。信息写得更多本身不加分；材料外扩写不算信息利用。
4. “先……再……”“一是……二是……”或正式连接词单次出现不算机械。只有成簇重复、没有真实关系的连接词链、每段同构、边界自证循环、固定三段式、口号式收束或主体宾语反复缺失时，才判机械，并至少引用两处证据。
5. 直接使用成本分为：G0 可直接采用；G1 只需不超过 3 处局部修正；G2 需要重组一段以上、补结构要素或系统删重；G3 事实、文种、格式或输出模式失真，需要重写。范围内的篇幅差异不改变成本等级。
6. 给出三稿排序。第一名相对第二名只可判“严格胜出”“小胜”或“难分”。“严格胜出”要求硬边界不低、至少两个非篇幅质量优势、其中至少一项来自可读性/自然度/直接使用成本，并且没有实质质量维度更差。
7. 若一稿事实更稳而另一稿更自然，或裁决只能依赖篇幅、偏好或单句差异，判为难分，不强行分胜负。

## 输出格式

- 硬边界表：A/B/C 的逐项 PASS/WARN/FAIL 与原文证据。
- 质量比较表：各维度谁更好或相当、文本证据、采用影响。
- 排序：第一、第二、第三；第一相对第二为严格胜出/小胜/难分。
- 风险：是否存在任何一稿独有的事实、状态、文种、格式、Prompt 或输出模式硬回退。

# 市图书馆自助借还设备试运行阶段报告

一、试运行基本情况

试运行期间，各点位按照既定安排开展设备运行、读者引导、故障记录和数据汇总等工作。6个点位累计完成自助借书48216册次、还书51703册次，借还合计99919册次；同期人工窗口完成借还32940册次。自助设备已成为馆内借还服务的重要补充，在分流人工窗口业务、方便读者办理借还手续、提高服务效率等方面发挥了积极作用。

从运行情况看，设备能够满足大部分读者日常借还需求，基本操作流程较为稳定。部分点位在高峰时段出现使用集中现象，读者对操作提示、图书识别、预约书取书位置等方面提出了改进意见，说明设备功能和现场服务仍需结合实际使用情况持续优化。

二、设备运行及技术保障情况

试运行期间，设备分别于4月12日、5月7日和6月18日出现短时离线，持续时间分别为11分钟、24分钟和9分钟。前两次通过重启设备恢复运行，第三次在网络恢复后自行恢复。三次离线均未造成长时间服务中断，但其共同原因目前尚未形成明确结论，设备离线预警、故障定位和应急处置机制仍需进一步完善。

为掌握设备记录准确性，6月对120笔业务记录进行抽查，其中113笔日志时间与实际业务时间一致，7笔日志时间与业务时间相差1至3分钟。抽查结果表明，设备业务记录总体准确，但部分日志时间存在小幅偏差，需继续关注系统时间同步、网络延迟和后台记录机制，必要时完善数据校核和异常记录办法。

三、读者反馈及服务改进需求

试运行期间共收到读者反馈87条。其中，操作提示不清31条，占比最高；少儿图书识别失败19条；预约书取书位置不易查找16条；希望延长夜间服务13条；其他意见8条。反馈主要集中在设备使用引导、特定类型图书识别、预约服务衔接和开放时段四个方面。

针对操作提示不清问题，部分设备界面和现场指引需要进一步简化表述、突出关键步骤，并根据读者实际操作情况调整提示顺序。针对少儿图书识别失败问题，应重点检查少儿图书标签、条码、封面材质及设备识别参数，区分设备原因和图书个体原因，形成具体排查记录。针对预约书取书位置不易查找问题，应进一步完善取书区域标识、楼层及书架提示，并优化线上查询与现场指引之间的衔接。对于延长夜间服务的意见，应结合晚间客流、设备使用量、现场管理和安全保障等情况进行评估，暂不作出扩大服务时段的决定。

四、培训和管理情况

5月完成两轮现场培训，共42人参加，培训内容主要包括设备基本操作、常见故障处置、读者咨询回应和运行记录填写等。通过培训，相关工作人员对设备使用流程和日常管理要求有了进一步了解。但从读者反馈和设备离线情况看，培训内容还需增加少儿图书识别异常、预约书取书指引、离线状态下的服务衔接以及故障上报等针对性内容。

下一阶段，应继续明确各点位日常巡查、问题登记、技术联系和数据报送要求，做到设备异常及时发现、及时记录、及时反馈。对于重复出现或影响读者使用的问题，应建立问题清单和跟踪销号机制，避免同类问题长期存在。

五、经费使用情况

试运行期间，设备租赁、网络和驻场服务三项支出合计27.8万元，预算为30万元，控制在预算范围内，预算执行率为92.67%。目前材料未显示新增采购决定，后续经费安排应结合延长试运行期间的实际需求、设备运行效果和馆务会审议意见另行研究。

六、下一阶段工作安排

6月26日，馆务会听取阶段汇报后提出，继续运行至8月31日，补充高峰时段测试后，再研究是否扩大设备数量。会议未形成新增采购决定。按照会议意见，7月至8月重点做好以下工作：一是记录晚间时段设备使用量、读者数量和业务类型，为评估服务时段提供依据；二是持续跟踪少儿图书识别情况，分类记录识别失败图书及处理结果；三是收集预约书取书路径反馈，优化标识和引导；四是完整记录设备离线、恢复时间、处置方式及网络状态，配合技术服务方查找共同原因；五是补充高峰时段测试，重点观察设备排队、人工引导、故障处理和服务承载情况；六是根据测试数据和读者反馈，形成阶段性分析材料。

目前，具体设备调整方案、责任分工和新增采购安排尚未确定。下一步应在不预设扩大数量的前提下，先完成延长试运行和补充测试，客观评估设备使用率、服务效率、故障情况、读者满意度及运行成本，再根据评估结果研究后续安排。试运行结束后，建议形成正式评估报告，提交馆务会审议。

二是部分操作提示不够清晰。试运行期间共收到读者反馈87条，其中反映操作提示不清31条，占反馈总数较高比例，说明部分读者在设备操作过程中仍存在理解和判断困难。该问题涉及设备提示内容和读者操作体验，需要在后续运行中结合具体反馈继续记录。

三是少儿图书识别情况有待改进。读者反馈中涉及少儿图书识别失败19条，反映设备对部分少儿图书的识别仍不够稳定。青少年分馆等点位的使用情况尤其需要关注，应通过后续高峰时段测试和业务记录，进一步掌握识别失败的具体表现。

四是预约书取书位置不易查找。共有16条反馈涉及预约书取书位置查询不便，表明部分读者在办理预约书取取或查找相关位置时，获取信息不够顺畅。该问题需要结合读者实际行走路径和现场反馈继续观察。

五是夜间服务需求尚未充分满足。共有13条反馈希望延长夜间服务，说明部分读者对延长设备服务时段存在需求。夜间使用量及其变化情况目前还需进一步记录，是否调整服务时段，仍应以继续运行期间形成的实际数据为依据。

三、经费使用情况

试运行期间，设备租赁、网络和驻场服务三项支出合计27.8万元，预算为30万元，实际支出未超过预算，较预算减少2.2万元。上述经费主要用于保障设备试运行及相关服务开展。后续是否扩大设备数量、调整设备配置或增加相关支出，需在本阶段试运行结束、补充测试并综合评估后另行研究。

四、馆务会意见及下一步安排

6月26日，馆务会听取阶段汇报，会议意见为继续运行至8月31日，补充高峰时段测试后，再研究是否扩大设备数量。会议未形成新增采购决定。根据会议意见，7月至8月拟重点记录晚间使用量、少儿图书识别情况、预约书取书路径反馈和离线情况，并结合高峰时段测试，进一步积累设备运行和读者使用数据。

下一阶段应围绕上述重点事项做好运行记录和情况汇总，保持对设备离线情况的持续关注，完整记录发生时间、持续时长及恢复情况；关注少儿图书识别表现，收集相关反馈；了解预约书取书路径中的实际问题；掌握晚间及高峰时段设备使用情况。同时，对业务时间与日志时间存在差异的记录继续进行抽查，观察该问题是否持续发生。

总体看，自助借还设备试运行期间完成了既定点位和设备范围内的运行，形成了一定数量的业务数据和读者反馈，设备也出现了离线、识别、提示和取书路径等方面的问题。鉴于部分问题仍需通过继续运行和补充测试进一步观察，现阶段不宜据此形成扩大设备数量或新增采购的结论。建议按馆务会意见继续运行至2026年8月31日，完成高峰时段及相关重点事项测试后，再根据实际数据和反馈研究后续安排。
