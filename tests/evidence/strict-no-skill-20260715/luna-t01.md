# 市图书馆自助借还设备试运行阶段报告

为检验自助借还设备运行效果，提升读者服务效率，市图书馆于2026年4月1日至6月30日开展自助借还设备试运行。试运行覆盖总馆、东城分馆、西城分馆、滨江分馆、青少年分馆、政务服务中心分馆6个点位，共配置设备18台。现将有关情况报告如下。

一、试运行基本情况

试运行期间，各点位按照既定安排开展设备运行、读者引导、故障记录和数据汇总等工作。6个点位累计完成自助借书48216册次、还书51703册次，借还合计99919册次；同期人工窗口完成借还32940册次。自助设备已成为馆内借还服务的重要补充，在分流人工窗口业务、方便读者办理借还手续、提高服务效率等方面发挥了积极作用。

从运行情况看，设备能够满足大部分读者日常借还需求，基本操作流程较为稳定。部分点位在高峰时段出现使用集中现象，读者对操作提示、图书识别、预约书取书位置等方面提出了改进意见，说明设备功能和现场服务仍需结合实际使用情况持续优化。

二、设备运行及技术保障情况

试运行期间，设备分别于4月12日、5月7日和6月18日出现短时离线，持续时间分别为11分钟、24分钟和9分钟。前两次通过重启设备恢复运行，第三次在网络恢复后自行恢复。三次离线均未造成长时间服务中断，但其共同原因目前尚未形成明确结论，设备离线预警、故障定位和应急处置机制仍需进一步完善。

为掌握设备记录准确性，6月对120笔业务记录进行抽查，其中113笔日志时间与实际业务时间一致，7笔日志时间与业务时间相差1至3分钟。抽查结果表明，设备业务记录总体准确，但部分日志时间存在小幅偏差，需继续关注系统时间同步、网络延迟和后台记录机制，必要时完善数据校核和异常记录办法。

三、读者反馈及服务改进需求

试运行期间共收到读者反馈87条。其中，操作提示不清31条，占比最高；少儿图书识别失败19条；预约书取书位置不易查找16条；希望延长夜间服务13条；其他意见8条。反馈主要集中在设备使用引导、特定类型图书识别、预约服务衔接和开放时段四个方面。

针对操作提示不清问题，部分设备界面和现场指引需要进一步简化表述、突出关键步骤，并根据读者实际操作情况调整提示顺序。针对少儿图书识别失败问题，应重点检查少儿图书标签、条码、封面材质及设备识别参数，区分设备原因和图书个体原因，形成具体排查记录。针对预约书取书位置不易查找问题，应进一步完善取书区域标识、楼层及书架提示，并优化线上查询与现场指引之间的衔接。对于延长夜间服务的意见，应结合晚间客流、设备使用量、现场管理和安全保障等情况进行评估，暂不作出扩大服务时段的决定。

四、培训和管理情况

5月完成两轮现场培训，共42人参加，培训内容主要包括设备基本操作、常见故障处置、读者咨询回应和运行记录填写等。通过培训，相关工作人员对设备使用流程和日常管理要求有了进一步了解。但从读者反馈和设备离线情况看，培训内容还需增加少儿图书识别异常、预约书取书指引、离线状态下的服务衔接以及故障上报等针对性内容。

下一阶段，应继续明确各点位日常巡查、问题登记、技术联系和数据报送要求，做到设备异常及时发现、及时记录、及时反馈。对于重复出现或影响读者使用的问题，应建立问题清单和跟踪销号机制，避免同类问题长期存在。

五、经费使用情况

试运行期间，设备租赁、网络和驻场服务三项支出合计27.8万元，预算为30万元，控制在预算范围内，预算执行率为92.67%。目前材料未显示新增采购决定，后续经费安排应结合延长试运行期间的实际需求、设备运行效果和馆务会审议意见另行研究。

六、下一阶段工作安排

6月26日，馆务会听取阶段汇报后提出，继续运行至8月31日，补充高峰时段测试后，再研究是否扩大设备数量。会议未形成新增采购决定。按照会议意见，7月至8月重点做好以下工作：一是记录晚间时段设备使用量、读者数量和业务类型，为评估服务时段提供依据；二是持续跟踪少儿图书识别情况，分类记录识别失败图书及处理结果；三是收集预约书取书路径反馈，优化标识和引导；四是完整记录设备离线、恢复时间、处置方式及网络状态，配合技术服务方查找共同原因；五是补充高峰时段测试，重点观察设备排队、人工引导、故障处理和服务承载情况；六是根据测试数据和读者反馈，形成阶段性分析材料。

目前，具体设备调整方案、责任分工和新增采购安排尚未确定。下一步应在不预设扩大数量的前提下，先完成延长试运行和补充测试，客观评估设备使用率、服务效率、故障情况、读者满意度及运行成本，再根据评估结果研究后续安排。试运行结束后，建议形成正式评估报告，提交馆务会审议。
