# 中文论文独立文献综述专项链

仅在最终交付物为独立文献综述时读取。用户给定的综述问题、范围、来源和格式优先；本文件不规定固定分类、文献数量、章节或篇幅。

## 事实与引用

- 只使用用户提供或已经核验的来源。作者、年份、题名、观点、方法、样本、数据和结论不得补造。
- 具体发现只来自实际读取的原文、摘要或用户材料；不能从题名和书目信息推断研究内容。
- 保持证据强度：相关、关联、反映、提及不得改写为影响、作用、导致或效果；来源未作因果识别时，不使用因果或效果表述。
- 每项来源归因都要能够回到对应材料。不同来源的观点不得相互移接，未核验信息不得写成确定事实。
- 默认不联网。用户明确要求检索时，保留来源信息和正文使用位置。
- 材料未覆盖某一问题时，只说明现有材料的覆盖范围，不能据此断言整个领域不存在相关研究。

## 由大至小写作

1. 锁定综述问题及范围，包括核心概念、研究对象、时间范围、地域或学科边界。
2. 建立内部来源清单，记录每项来源实际支持的观点、方法、材料、结论及限制。
3. 根据来源之间的真实关系选择主题、观点、方法、时间或其他组织轴；不为形式整齐强行分类。
4. 按“综述问题 -> 分组任务 -> 组内共识与分歧 -> 证据基础 -> 局限或待研究问题 -> 全文结论”形成提纲。
5. 每段按“比较问题 -> 多项来源证据 -> 异同分析 -> 本段结论”组织，避免逐篇摘要连续罗列。
6. 研究不足、争议和空缺只能由已核验材料支持；作者自己的判断保持为分析判断，不冒充来源结论。
7. 结尾只归纳正文已经完成的综合，不新增来源、事实或研究发现。

## 正文后构造建议

用户未限制只输出正文时，可简要列出：

- `可补充材料`：现有综述范围内缺少的来源类型或证据。
- `可补充论点`：可以进一步比较但当前材料不足的方向。
- `可展开论点`：已有多项来源能够继续综合的位置。

只列少量、具体、可操作的建议，不编写虚构来源、观点、分歧或领域空白。用户要求只输出正文时，不附构造建议。

## 由小至大复核

- 段落：每项归因是否对应正确来源，比较结论是否超出材料。
- 小节：是否围绕同一分组任务形成综合，是否退化为逐篇罗列。
- 全文：题目、范围、分类轴、来源覆盖和综合结论是否一致；引用序号和参考文献是否对应；是否把材料未覆盖误写成领域空白。
- 交付：是否遵守用户格式、只输出正文、只输出改后稿或只审不改等要求；是否残留过程说明、规则自述或未完成占位。
